#### 1. Roles

**Administrator:** Owner who manages the system

**Member:** Registered participant who is involved in the trading

#### 2. Operation actions

**read:** Retrieve the status details of a Document Type.

#### 3. Permissions


|      Action                      | Administrator       | Member            |
|----------------------------------|---------------------|-------------------|
| request-for-status:read  | ✓                   | ✓                |

**Administrator:** `read` access allows the administrator to request for status.

**Member:** `read` access allows the administrator to request for status.
